Mi Página para Implantación de Aplicaciones Web

Bienvenido/a

Implantación de Aplicaciones Web es una asignatura orientada a que el alumno desarrolle las competencias necesarias para desplegar, mantener y gestionar aplicaciones web en distintos entornos, tanto locales como en la nube. Se aprende de forma práctica cómo instalar servidores, configurar servicios y publicar proyectos reales.

¿Qué objetivos tiene esta asignatura?

  • Comprender el funcionamiento de los principales servidores web, como Apache y Nginx.
  • Instalar y configurar aplicaciones web con lenguajes como HTML, CSS, JavaScript y PHP.
  • Gestionar bases de datos (MySQL, MariaDB) y conectar aplicaciones web a ellas.
  • Trabajar con herramientas como XAMPP, VirtualBox y GitHub para simular entornos reales.
  • Aplicar buenas prácticas de seguridad y gestión de permisos en servidores.
  • Automatizar tareas y realizar copias de seguridad para proteger la información.
  • Desplegar proyectos web y ponerlos en producción usando servicios gratuitos como GitHub Pages.

Ejemplo de prácticas que se realizan

  • Instalación y configuración de un servidor web (Apache2) en Windows y Linux.
  • Publicación de una web estática usando HTML y CSS en GitHub Pages.
  • Montaje y prueba de bases de datos con MySQL y conexión desde PHP.
  • Creación de máquinas virtuales para simular clientes y servidores.
  • Documentación de cada paso con capturas y explicación técnica.

En esta página iré publicando los diferentes trabajos y proyectos que vaya desarrollando durante el curso, mostrando el proceso técnico y los resultados obtenidos.
